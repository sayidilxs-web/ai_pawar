## 🎯 বিবরণ

NEXUS AI হলো একটি শক্তিশালী AI অ্যাসিস্ট্যান্ট যা বাংলায় কথা বোঝে এবং বাংলায় কথা বলে। এটি আপনার কম্পিউটারের স্ক্রিন দেখতে পারে, মাউস ও কিবোর্ড কন্ট্রোল করতে পারে, এবং আপনার সাথে স্বাভাবিক কথোপকথন করতে পারে।

### 🧠 NEXUS Neural Network
এই প্রজেক্টে একটি সম্পূর্ণ Neural Network সিস্টেম রয়েছে যা JSON ফাইল থেকে সরাসরি জ্ঞান লোড করে এবং Smart Learning পাইপলাইনের সাথে সংযুক্ত।

### 🧠 Neural Network সিস্টেম
- **NEXUS Neural Network Core**: Forward/Backward propagation, Layers, Activations
- **Smart Learning System**: Adaptive learning rate, Pattern recognition, Self-improvement
- **Knowledge Base System**: JSON ফাইল থেকে সরাসরি জ্ঞান লোড
- **Transaction Pipeline**: Multi-stage data processing
- **No Training Required**: সরাসরি ডাটা দিয়ে কাজ করে

### ⚡ LIGHT MODE (চোখের পলকে)
- **Light Cache**: ইনস্ট্যান্ট ডেটা অ্যাক্সেস
- **Light Scraper**: বিনামূল্যে রিয়েল-টাইম ডেটা (News, Weather, Crypto)
- **Auto Update**: স্বয়ংক্রিয় আপডেট (প্রতি ৫-৩০ মিনিট)
- **Smart Search**: দ্রুত প্যাটার্ন ম্যাচিং
- **NEXUS Brain**: সবকিছু একসাথে কানেক্ট - **API ছাড়াই!**

## 🚀 ইনস্টলেশন

### প্রয়োজনীয়তা
- Node.js 18+
- npm বা yarn
- Windows 10+, macOS 10.15+, বা Linux

### স্টেপ ১: রিপোজিটরি ক্লোন করুন
```bash
git clone https://github.com/rifat1122ddf-commits/ai.git
cd ai
```

### স্টেপ ২: ডিপেন্ডেন্সি ইনস্টল করুন
```bash
npm install
```

### স্টেপ ৩: জেমিনি API কী পান
1. [Google AI Studio](https://makersuite.google.com/app/apikey) যান
2. API কী তৈরি করুন
3. অ্যাপে সেটিংসে API কী যোগ করুন

### স্টেপ ৪: অ্যাপ চালান
```bash
npm start
```
